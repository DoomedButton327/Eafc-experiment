How to use in PaperCopy

  1. Download the exported image using the button below
  2. Import the image into PaperCopy on your tablet
  3. Set PaperCopy to display the image at 100% — no pinch zoom
  4. The drawing will now be at exact 1:1 real-world scale on your screen
  5. Use the Freeze feature and trace away!
